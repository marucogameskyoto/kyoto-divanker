# Kyoto Divanker

Static HTML game package for GitHub Pages.

## Files

- `index.html`
- `assets/kyoto-divanker-logo.svg`
- `assets/stage01-mountain-anomaly.png`
- `assets/stage02-shrine-anomaly.png`
- `assets/stage03-office-anomaly.png`

## Publish With GitHub Pages

1. Create a new GitHub repository.
2. Upload all files in this folder, keeping `index.html` at the repository root.
3. Open repository `Settings`.
4. Open `Pages`.
5. Under `Build and deployment`, choose:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/ (root)`
6. Save.
7. Wait for GitHub Pages to show the published URL.

## Embed In Google Sites

1. Open your Google Site.
2. Choose `Insert`.
3. Choose `Embed`.
4. Paste the GitHub Pages URL.
5. Resize the embedded frame.
6. Publish the Google Site.
